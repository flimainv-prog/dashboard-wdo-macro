# 📈 Dashboard WDO Macro v2

Dashboard em tempo real para análise macro do WDO (Dólar Futuro B3) com foco em previsão de movimento usando indicadores macro.

## 🎯 Objetivo

Auxiliar traders a tomar decisões baseadas em:
- **Linha Azul v2**: Índice composto macro (USD/MXN + DXY + VIX + TNX + USDZAR)
- **Spread Verde/Vermelha**: Quantidade de ativos comprados vs. vendidos
- **RoRo Score**: Indicador de apetite por risco global
